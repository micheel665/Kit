Function: kit.load(path, isurl)

Load a file or webpage.

Response: Fetched data response (string). Returns false if an error occurs.

Parameters

path (required)

The path to the file including the file name.

Example

print( kit.load("/path/to/file/hello.txt") )

Result

The contents of hello.txt .

isurl (optional)

Boolean (true). If this parameter is provided, the given path will be processed as a webpage. Please note that this will only work if the HTTP API has been enabled, and the requested webpage domain is whitelisted.

Example

print( kit.load("http://www.timeapi.org/cet", true) )

Result

The contents of the webpage (in this case: http://www.timeapi.org/cet).

Examples

This snippet will print a random cat fact using the catfacts API website. 
print( kit.load("http://catfacts-api.appspot.com/api/facts", true) )
