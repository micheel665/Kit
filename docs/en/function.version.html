Function: kit.version(latest)

Returns the version of your Kit distribution, or the latest version of Kit if using the latest parameter.

Response: String response.

Parameters

latest (optional) Kit 1.2.0 >

If this parameter is provided, the function will return the latest stable version of Kit using the HTTP API, instead of returning its own version.

Example

print( kit.version("latest") )

Result

The latest Kit version, eg: 1.1.0

Examples

This snippet will compare the latest version of Kit with its own version. 
myVersion = kit.version()
latestVersion = kit.version("latest")

if myVersion == latestVersion then
  print("My Kit is up to date!")
else
  print("My Kit version is diffrent from the newest version!")
end
