Function: kit.isAdvanced()

Checks whether the computer is advanced or not.

Response: Boolean response (true or false).

Parameters

None.

Examples

This snippet checks if the computer is advanced. 
if kit.isAdvanced() then
  print("I'm an advanced computer! I support colors!")
else
  print("I'm just a regular computer.")
end
