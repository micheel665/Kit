Function: kit.save(data, path, method)

Save given data to a new or existing file.

Response: Boolean (true or false).

Parameters

data (required)

The string you want saved to a file.

path (required)

The path to the (new) file, including the file name.

method (required)

The method you want for writing to the file. This can be:
  • 'w' to open the file for writing, and remove any existing data on file open.
  • 'a' to open the file for writing, but keep existing data and append the new data to the end of the file.

Examples

This snippet will demonstrate the 'w' and 'a' saving methods. 
data = "Hello World!\n"
result = kit.save(data, "/hello.txt", "w")
if result then
  print("File was saved!")
end
data = "Hello there stranger!"
result = kit.save(data, "/hello.txt", "a")
if result then
  print("File was saved!")
end
contents = kit.load("/hello.txt")
print(contents)
The output will be: 
File was saved!
File was saved!
Hello World!
Hello there stranger!
