Function: kit.explode(string, separator)

Explode a string with given separator.

Response: Table response.

Parameters

string (required)

The string you want exploded.

separator (required)

The separator to split the string with.

Examples

This snippet will split a string into a table using the explode function. 
string = "first,second,third,fourth"
tableResult = kit.explode(string, ",")
print( kit.dump(tableResult) )
The table's contents will be: 
{
  [1] = "first",
  [2] = "second",
  [3] = "third",
  [4] = "fourth"
}
